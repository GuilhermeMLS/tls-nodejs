Relatório do Trabalho de Tópicos Avançados em Redes de Computadores

Guilherme Morais Lopes dos Santos (GRR20163043) | Cristopher Luiz Assad Carcereri (GRR20163073)

Objetivo

Desenvolver uma comunicação entre cliente e servidor usando o protocolo TLS (Transport Layer Security) e demonstrar seu funcionamento.

Resumo do funcionamento do sistema

A aplicação foi desenvolvida em JavaScript, utilizando o ambiente Node.js, e é composta por um cliente (tslClient.js) e um servidor (tslServer.js). O script setup.sh automatiza a geração dos certificados e chaves criptográficas utilizando o OpenSSL.
Utilizamos o pacote interno tls do Node.js (disponível neste link) para criar um servidor TLS utilizando os certificados e chaves gerados pelo OpenSSL.

Relatório e funcionamento

Escolhemos a modalidade de vídeo para apresentação do nosso relatório. O vídeo explica o funcionamento do TLS, em seguida demonstra como implementar uma comunicação segura entre cliente e servidor usando JavaScript e Node.js através do projeto desenvolvido. O vídeo está disponível neste link. Para executar o programa, basta seguir os seguintes passos:

  1. Instalar o CLI do OpenSSL;
  2. Executar o script setup.sh preenchendo os dados requisitados (deve gerar o diretório certificates na raíz do projeto);
  3. Executar o servidor usando o comando $ node tlsServer.js (deve imprimir a porta de escuta do servidor no stdout);
  4. Em outra janela to terminal, executar o cliente usando o comando $ node tlsClient.js (estabelecerá uma conexão segura com o servidor, receberá dados e imprimirá logs no stdout).
